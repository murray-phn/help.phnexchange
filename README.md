# Setup development environment
1. Download python
2. `pip install mkdocs`
3. `pip install mkdocs-material`

# How to edit
1. Create or update the .md files within /src
2. Run `mkdocs serve` to show the changes locally

# How to release
1. Run `mkdocs build`
2. Commit & push the github repository
